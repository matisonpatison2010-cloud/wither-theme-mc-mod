Wither Theme Mod (Fabric 1.21.1)

HOW TO BUILD:

1. Install JDK 21
2. Open this folder in IntelliJ IDEA (recommended)
3. Let Gradle import
4. Run: gradlew build
5. Your mod jar will appear in build/libs/

IMPORTANT:
Place your custom music file here before building:

src/main/resources/assets/withertheme/sounds/wither_theme.ogg

The file MUST be .ogg format.
